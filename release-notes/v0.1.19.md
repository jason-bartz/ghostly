# v0.1.19

Smoother overlays, clearer progress, and a bunch of reliability fixes.

## New

- **Six overlay positions.** Pin the recording overlay to any screen corner — top-left, top-center, top-right, bottom-left, bottom-center, bottom-right — instead of just top or bottom. Set it under Settings → Advanced → Overlay Position. Existing setups keep their current top/bottom placement.
- **Download progress shows MB and ETA.** Model downloads and app updates now display how much is done, download speed, and time remaining, so a stalled download is obvious.
- **Simpler tag filtering.** Hover any tag in the history filter bar for an inline × that deletes it from every note at once.

## Refined

- **Cleaner dictation output.** Post-processing rarely leaks "Here's the cleaned text:" or similar preambles anymore. The built-in prompts (Improve Transcriptions, Developer, Email, Casual, Structured Notes) got stricter output rules.
- **Auto-tagging is grounded in what you said.** Auto-applied tags now always require the tag word — or a prefix of it — to actually appear in the transcription, so you stop seeing semantically-adjacent tags the model guessed.
- **Per-App Profiles page explains itself.** A new intro clarifies that profiles override your active refinement prompt (and vocabulary / provider) when a specific app is in focus.

## Accessibility

- Modals (update, paywall, license agreement) now trap keyboard focus and restore it when dismissed. Escape closes dismissible modals.
- API key and base URL inputs are labeled for screen readers.
- The EULA now shows a scroll-progress bar so you know how much of the agreement is left before you can accept.

## Reliability

- The app no longer crashes on startup if a tray icon asset is missing, if the tray fails to build, or if Unix signal handlers can't be installed — those failures log clearly and Ghostly keeps running.
- If your settings file can't be parsed after an upgrade, the raw contents are now preserved under a timestamped backup key in `settings_store.json` instead of being wiped. You can recover shortcuts, prompts, and license state from the backup.
